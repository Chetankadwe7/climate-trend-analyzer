# 🌍 Climate Trend Analyzer

## 📌 Overview
This project analyzes climate data and builds a complete machine learning pipeline for:
- Trend Analysis
- Anomaly Detection
- Forecasting

It helps understand long-term environmental patterns and detect unusual climate behavior.

---

## 🔄 Project Workflow

1. Load / Generate Data  
2. Data Cleaning  
3. Feature Engineering  
4. Trend Analysis  
5. Anomaly Detection  
6. Model Training (Forecasting)  
7. Save Outputs  
8. Visualization  

---

## 🚀 Features

- 📊 Climate trend analysis (Temperature & Rainfall)
- 🚨 Anomaly detection using statistical methods
- 🔮 Forecasting using Machine Learning
- 📁 Structured output generation
- 📈 Multiple visualizations

---

## 🛠 Tech Stack

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn
